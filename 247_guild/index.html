冒険者ギルド「帝国の鉢巻」亭

お仕事はいつでもお待ちしております～

「帝国の鉢巻」亭は、帝都で最も信頼されている冒険者ギルドの一つ、だと思います。たぶん。
腕に覚えのある方、一攫千金を夢見る方、どなたでも歓迎いたします～。依頼をこなして、有名になってみませんか？

ギルド概要

私たち「帝国の鉢巻」は、港湾都市ハーヴェスの外れに拠点を構える冒険者ギルドです。建物は魔動機文明時代の遺跡をリフォームしたもので、その地下には未だに謎の区域が広がってるみたいですよ。ギルドマスター「ミカ・ニシナ」さんの指揮のもと、ベテランから私みたいな新人まで、色々な冒険者さんが集まって、毎日いろんな依頼に挑んでいます。最近は、討伐したヒドラを原料にした珍しいお酒も造り始めたみたいで、新しい収入源として期待されてる…らしいです。
それに、最近はたくさんの新人冒険者さんが来てくれて、ギルドはとっても賑やかです。でもその一方で、ニシナさんのサービス精神が旺盛すぎて、経営はいつも赤字だって聞いてます…。大丈夫なんでしょうか、このギルド……。

スタートダッシュキャンペーン

キャンペーンに対応している依頼には、このロゴが付いてるみたいなので、チェックしてみてくださいね！見逃しちゃダメですよ～！

こ、怖い…！何が出るかは開けてのお楽しみ、だそうです…。

ギルドマスターの発案で始まった、新人冒険者さん向けの特別キャンペーンです。新しくギルドに入った冒険者さんは、この怪しげな箱からくじを引くことができるんですよ。くじの番号に応じて、冒険に役立つ（かもしれない）色々なアイテムがもらえるみたいです。ニシナさん曰く「赤字覚悟！出血大サービス！」とのことですが、本当なんですかね…？これまでには「陽光の魔符」や「酒の種」、「スモークグラファー」といった珍しい品々が出たという記録も残ってるみたいです。

ギルド内で囁かれる不穏な噂…？

「ギルドの財政は常に火の車と聞く。この過剰なサービスは、我々の報酬から不当に捻出されているのではないか？」…なんて書かれたビラがギルドの隅に落ちてました。ニシナさんの浪費癖が、ついに一部の冒険者さんの不満を買っちゃったみたいですね…。

施設案内

併設酒場

ギルド1階に併設された酒場です。特に名前はないみたいですけど、冒険者の皆さんの憩いの場になってます。名物は「山盛りポテト」と独特の風味を持つ「ルートビア」ですが、ポテトの塩加減は日によって結構違うみたいですね…。最近はニシナさんの気まぐれで、自家製ワインが振る舞われることもあるって聞いてます。

宿泊施設

ギルドの2階以上は冒険者さん向けの宿泊施設になっています。依頼を受けている冒険者さんは無料で泊まれるんですよ。それ以外の方も、採算を考えてるとは思えないくらい安い料金で泊まれるので、たくさんの冒険者さんの拠点になってるみたいです。

訓練場・工房

中庭には模擬戦用の案山子や的が置いてある訓練場があります。あと、地下には魔動機技師のリリさん専用の工房があって、冒険者さんたちの武具の修理や改造を全部やってくれてるんです。ニシナさんは将来、過去の依頼データをもとにした擬似的な魔域での訓練施設も作りたいって言ってるみたいですよ。

地下醸造所と未踏遺跡

ギルドの地下には、ニシナさんこだわりのワインや、ヒドラを原料にした特殊なお酒を造って熟成させてる醸造所があるんです。さらにその奥、地下3階から先はまだ誰も入ったことがない未探索の遺跡が広がっていて、ギルド最大の謎になってるみたいです。ちょっと怖いですよね…。

ギルドスタッフ

ミカ・ニシナ

人間 / ギルドマスター

「帝国の鉢巻」亭のギルドマスターです。遠いサバナイ地方のご出身だそうですが、詳細はあまり語られません。赤字経営に頭を悩ませつつも、自ら最前線で戦う実力者です。新人向けのキャンペーンで採算を度外視してしまうなど、そのサービス精神が赤字の主な原因だと伺っております。地下でオリジナルワインを熟成させるなど、多趣味な一面も持ち合わせています。

ゾロメ・ベストマン

人間 / 事務員 (新卒)

ギルドの事務員を務めております、ゾロメ・ベストマンです。魔法学園を卒業して、最近ギルドに入ったばかりの新卒です。実務経験はまだ浅いですが、依頼の受付から報酬の支払いまで、精一杯頑張りますので、どうぞよろしくお願いいたします～。あ、あと、天地使い…ジオマンサーっていうのもやっています。まだまだ半人前ですけど、いつか皆さんのお役に立てるように頑張りますね！

ラズヒェル・リリベラード

人間 / 魔動機技師

ギルドお抱えの優秀な魔動機技師です。皆からはリリさんって呼ばれているようですね～。地下の工房で冒険者が持ち込む魔動機の修理や改造を一手に引き受けています。自身の発明品のテストを依頼してくることもあります。

ポポロ・リック

グラスランナー / 先輩冒険者

新人冒険者の案内役を務めることが多い、面倒見の良い先輩です。斥候としての腕は一流で、数々の修羅場をくぐり抜けてきたと伺っております。

バイトさん

ウィークリング / ギルド職員 (バイト)

最近ギルドで働き始めた、名前を教えてくれない職員さんです。突発的な出来事に弱く、すぐにパニックになってしまいますが、給金のためならと必死に業務に取り組んでいます。ギルドの財政状況を常に心配している苦労人でもあります。

ジン (Jin)

種族不明 / ギルドの…？

ギルドの片隅にいつの間にか現れる、なんかうざったい謎の男です。こいつ、「もしも全滅したときにはなんとかするから」とか言ってますけど、その真意は定かではありません。カジノに入り浸っては有り金を溶かしたり、釣りでとんでもないものを釣り上げてきたり…本当に何を考えているのか分からなくて、正直関わるのがだるいです。その正体、目的は誰も知りません。（…というか、知りたくもないですね）

注目の冒険者さん

当ギルドでは、普通の知名度や冒険者ランクとは別に、個人の戦闘能力や依頼達成への貢献度を総合的に評価する独自の指標（1～15段階）を設けているみたいです。これのおかげで、それぞれの実力に合った依頼を紹介しやすくなってるんですね。ちなみに、今ギルド内で最高段階に到達している冒険者さんはいるみたいですけど、まだ誰も到達したことがない16段階目に到達した人はいないそうです。いつか現れるんでしょうか…！

ケイル・ブロンコリリー

街の施設

武具屋「鋼の魂」

ハーヴェスで一番の品揃えを誇る武具屋です。店主は頑固ですが、腕は確かですよ。

道具屋「何でも屋」

回復薬から魔動機の部品まで、冒険に必要なものは大体ここで揃います。たまに掘り出し物があるとか…？

危険地帯

ゴブリンの森

ギルドから南に位置する森です。最近ゴブリンが棲み着いて、旅人を襲っているとの報告が多数寄せられています。新人冒険者さんは特に注意してくださいね。

忘れられた廃墟

街の東にある古い遺跡です。アンデッドが出没するという噂なので、準備を万全にしてから向かってください。

特殊な場所

回復の泉

森の奥深くにあると言われている泉です。その水を飲むと傷が癒えると言われていますが、見つけるのは大変みたいです。

隠された宝箱

廃墟のどこかに、古代の宝が眠る宝箱があるという噂です。罠が仕掛けられている可能性が高いので、腕利きの斥候さんと一緒じゃないと危ないですよ！

ギルドのウワサ

酒場で冒険者さんたちが話しているウワサを、こっそりまとめてみました。信じるか信じないかは、あなた次第ですよ…？
